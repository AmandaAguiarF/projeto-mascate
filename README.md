# Projeto Mascate

Projeto desenvolvido para a disciplina de Coding da Faculdade Senac PE. Foi proposto o desenvolvimento de um sistema de Gerenciamento de Pedidos para o Mascate. 

## Tecnologias Utilizadas

### Backend:
- Typescript
- Express
- TypeORM
- Node.js  
- MySQL

### Frontend:
- HTML
- CSS
- Angular
- Bootstrap
- Angular Material
